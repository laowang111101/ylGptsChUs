# 图书借阅推荐系统 python188

## 项目概述

本系统是基于Python开发的图书借阅推荐系统。采用Django框架结合MySQL数据库进行后端开发，使用Vue.js构建前端界面，实现了一套完整的图书借阅流程以及管理功能。通过用户协同过滤算法，为读者推荐合适的图书。

## 技术栈

- 后端：Python + Django
- 数据库：MySQL
- 前端：Vue.js

## 用户-图书评分矩阵与推荐逻辑

### 用户-图书评分矩阵构建

通过 `build_user_item_matrix()` 函数构建用户-图书评分矩阵。评分标准如下：

- 收藏书籍的评分为 1
- 心愿书籍的评分为 0.5
- 其他书籍的评分默认为 0

### 计算用户相似度

使用余弦相似度 (`cosine_similarity`) 计算用户之间的相似度。

### 推荐逻辑

根据用户相似度矩阵，为目标用户推荐相似用户所喜欢的书籍。推荐书籍限制为最多 12 本，且为目标用户未曾收藏或添加到心愿单的书籍。

### 协同过滤

基于用户的协同过滤，推荐与目标用户相似的其他用户喜好的图书。
